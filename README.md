# Boris Oskolkov — ML Researcher Portfolio

Personal portfolio site built with Vite + React + TypeScript + Tailwind CSS, deployed on GitHub Pages.

**Live:** https://borisoskolkov.github.io/

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The build output goes to `dist/`.

## Deploy

Pushes to `main` automatically build and deploy to GitHub Pages via the workflow in `.github/workflows/deploy.yml`.

## Stack

- React 19 + TypeScript
- Vite 7 (build tool)
- Tailwind CSS 4
- Radix UI primitives
- Wouter (client-side routing)
- TanStack Query
